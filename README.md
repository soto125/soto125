## 📱 Proyectos Destacados

### 🛡️ Sistema de Gestión SST (Seguridad y Salud en el Trabajo)
*Proyecto desarrollado durante prácticas profesionales.*
- **Descripción:** Plataforma para la automatización de reportes de riesgos, seguimiento de normativas y gestión de incidentes laborales.
- **Logro:** Digitalización completa de procesos físicos, facilitando el cumplimiento legal en tiempo real.
- **Stack:** Node.js, Express, PostgreSQL, React.

### 🚗 ParkApp
*Solución inteligente para gestión de estacionamientos.*
- **Funcionalidades:** Autenticación, sistema de reservas, pagos integrados y geolocalización.
- **Key Feature:** Integración fluida con Google Maps y Waze para navegación directa.
